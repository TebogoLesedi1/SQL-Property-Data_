# SQL Property Data Analysis

## 📋 Project Overview

This project involves analyzing a large dataset of residential properties across all nine provinces of South Africa using SQL queries. The dataset simulates a real estate platform similar to Property24, focusing on writing **read-only queries** to answer business-style questions.

### 🎯 Project Goals

The goal is to explore, filter, and analyze property data using SQL. Key objectives include:

- Finding top properties by price or size  
- Filtering properties by location or type  
- Sorting and presenting data in an organized manner  
- Demonstrating the ability to interpret real-world business questions into SQL queries

## ✨ Key Features

- **SELECT queries** with WHERE conditions for precise filtering  
- **Sorting results** using ORDER BY for organized output  
- **Limiting results** using TOP or equivalent for performance optimization  
- **Efficient analysis** of large datasets  

## 🛠️ Tech Stack

- **Database**: SQL Server / MySQL
- **Tools**: SQL Server Management Studio (SSMS), MySQL Workbench
- **SQL Commands**: SELECT, WHERE, ORDER BY, TOP

## 🚀 How to Run

1. Open your SQL client (SQL Server Management Studio, MySQL Workbench, etc.)  
2. Connect to the provided database containing the property dataset  
3. Run the queries one by one from `queries.sql`  
4. Review the results for accuracy and readability
